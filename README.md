# ⌚ WatchStore - Mini E-Commerce Store

A PHP & MySQL mini e-commerce store built with XAMPP.

## Technologies Used
- HTML, CSS, Bootstrap 4
- JavaScript
- PHP (OOP with CRUD class)
- MySQL / phpMyAdmin
- XAMPP

## How to Run
1. Install XAMPP
2. Import watchstore.sql into phpMyAdmin as watchstoredb
3. Copy project into htdocs/watchstore/
4. Visit http://localhost/watchstore/
